# Data Sources

## Public Safety
- Source: City of Los Angeles Open Data
- Dataset: Crime Data
- Update Frequency: Daily
- Main Fields:
  - Date
  - Location
  - Crime Type
  - Neighborhood

## Mobility
- Source: Public Traffic / Transportation APIs
- Update Frequency: Hourly
- Main Fields:
  - Timestamp
  - Traffic Volume
  - Area / Corridor

## Weather
- Source: OpenWeather / NOAA
- Update Frequency: Hourly
- Main Fields:
  - Temperature
  - Rain
  - Weather Conditions
