# Data Sources

## English

### Public Safety
- Source: City of Los Angeles Open Data
- Dataset: Crime Data
- Update Frequency: Daily
- Main Fields:
  - Date
  - Location
  - Crime Type
  - Neighborhood

### Mobility
- Source: Public Traffic and Transportation APIs
- Update Frequency: Hourly
- Main Fields:
  - Timestamp
  - Traffic Volume
  - Area or Corridor

### Weather
- Source: OpenWeather API or NOAA
- Update Frequency: Hourly
- Main Fields:
  - Temperature
  - Rain
  - Weather Conditions

## Português

### Segurança Pública
- Fonte: Open Data da Cidade de Los Angeles
- Conjunto de Dados: Dados de Crimes
- Frequência de Atualização: Diária
- Campos Principais:
  - Data
  - Localização
  - Tipo de Crime
  - Bairro

### Mobilidade
- Fonte: APIs públicas de tráfego e transporte
- Frequência de Atualização: Horária
- Campos Principais:
  - Timestamp
  - Volume de Tráfego
  - Área ou Corredor

### Clima
- Fonte: API OpenWeather ou NOAA
- Frequência de Atualização: Horária
- Campos Principais:
  - Temperatura
  - Chuva
  - Condições Climáticas
