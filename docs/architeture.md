# System Architecture

## English

The system follows a modular and layered architecture designed to support data
integration, analysis, and visualization.

### Architecture Layers
1. Data ingestion from public APIs
2. Data cleaning and normalization
3. Data storage in a structured database
4. Analytical layer for metrics and aggregation
5. Visualization through dashboards
6. Rule-based recommendations and short-term predictions

### Data Flow
APIs → Ingestion Scripts → Processed Data → Database → Analytics → Dashboard

---

## Português

O sistema segue uma arquitetura modular e em camadas, projetada para suportar a
integração, análise e visualização de dados.

### Camadas da Arquitetura
1. Ingestão de dados a partir de APIs públicas
2. Limpeza e normalização dos dados
3. Armazenamento em banco de dados estruturado
4. Camada analítica para métricas e agregações
5. Visualização por meio de dashboards
6. Recomendações baseadas em regras e predições de curto prazo

### Fluxo de Dados
APIs → Scripts de Ingestão → Dados Processados → Banco de Dados → Análises → Dashboard
